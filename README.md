# NetworkConnection
Very Simple TCP Socket Implementation.

## Build
If you want to build for [Ev3Dev Platform](http://www.ev3dev.org/), you have to configure the project with this :
```bash
mkdir -d build
cd build/
qmake "CONFIG+=arm" ..
```

Otherwise, you just have to configure like this :
```bash
mkdir -d build
cd build/
qmake ..
```
